# 项目介绍：基于 AutoGen 的多 Agent 活动策划系统

## 1. 项目背景
传统活动策划往往依赖人工经验，容易出现信息不完整、讨论效率低、方案不够系统的问题。为提高策划效率，本项目尝试引入多 Agent 协作机制，让不同角色的 AI 共同参与活动方案生成。

## 2. 项目目标
构建一个能够自动完成以下流程的系统：
1. 根据用户输入生成筹备会议方案
2. 由多个不同立场的 Agent 模拟讨论
3. 自动输出可执行的活动策划书
4. 通过可视化前端展示整个推理与讨论过程

## 3. 核心创新点
- **多 Agent 协作**：不是单模型直接出结果，而是分角色讨论
- **三阶段流程**：筹备会议 → Agent 讨论 → 策划书汇总
- **动态发言机制**：采用 `SelectorGroupChat` 选择最合适的下一位发言者
- **可视化交互**：支持实时流式展示、虚拟角色、高亮发言、决议卡、历史任务回放

## 4. 技术方案
- **框架**：AutoGen
- **模型接入**：DeepSeek API
- **后端**：FastAPI
- **前端**：HTML / CSS / JavaScript
- **交互方式**：NDJSON 流式事件推送

## 5. 系统价值
本系统可用于：
- 校园活动策划
- 分享会/宣讲会设计
- 小型赛事或培训活动准备
- 团队内部协同讨论辅助

## 6. 当前成果
目前系统已经可以：
- 自动生成活动筹备会议方案
- 模拟多角色 Agent 讨论
- 输出活动策划书
- 在前端实时展示 Agent 发言与 moderator 决议卡

## 7. 后续扩展方向
- 引入真正的 3D 虚拟角色
- 增加语音播报
- 支持更多活动模板
- 增加争议面板与决议自动汇总
